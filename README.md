# MEGAMESTORE: Game Distribution Management System

📌 Project Overview
MEGAMESTORE is a comprehensive system analysis and design project for a game distribution platform. The system is designed to bridge the gap between game developers and gamers, facilitating seamless transactions, digital rights management, and community engagement. This project covers the full lifecycle of system analysis, including requirements gathering, process modeling (DFD), and database design (ERD).

🚀 Key Modules
- User Management: Registration, authentication, and profile management for both gamers and developers.

- Game Distribution: Tools for developers to upload, manage, and price their digital products.

- Transaction System: Secure purchasing workflow, including cart management and digital key activation (via platforms like G2A/Steam).

- Library Management: A centralized digital library for users to track and access their purchased games.

- Reporting & Analytics: Sales tracking and system performance monitoring for administrators.

🛠 Methodology & Tools
- System Analysis: Data Flow Diagrams (DFD) - Level 0 to Level 2 (Gane-Sarson notation).

- Database Design: Entity-Relationship Diagrams (ERD) and relational schemas.

- Documentation: Structured technical reporting following university standards.

# MEGAMESTORE: Hệ Thống Quản Lý Mua Bán Nền Tảng Phân Phối Game

📌 Tổng quan dự án
MEGAMESTORE là đồ án phân tích và thiết kế hệ thống cho một nền tảng phân phối trò chơi điện tử trực tuyến. Hệ thống được xây dựng nhằm kết nối giữa nhà phát triển game và game thủ, hỗ trợ các quy trình giao dịch, quản lý bản quyền số và tương tác cộng đồng. Dự án tập trung vào quy trình phân tích hệ thống toàn diện, từ khảo sát yêu cầu, mô hình hóa quy trình nghiệp vụ (DFD) đến thiết kế cơ sở dữ liệu (ERD).

🚀 Các phân hệ chính
- Quản lý người dùng: Đăng ký, đăng nhập và quản lý hồ sơ cho cả người chơi và nhà phát triển.

- Phân phối Game: Công cụ dành cho nhà phát triển để tải lên, quản lý thông tin và định giá sản phẩm.

- Hệ thống giao dịch: Quy trình mua hàng bảo mật, quản lý giỏ hàng và kích hoạt mã game (tương tự G2A/Steam).

- Quản lý thư viện: Thư viện kỹ thuật số tập trung giúp người dùng theo dõi các game đã sở hữu.

Báo cáo & Thống kê: Theo dõi doanh thu và hiệu suất hệ thống dành cho quản trị viên.

🛠 Phương pháp & Công cụ
- Phân tích hệ thống: Sơ đồ dòng dữ liệu (DFD) từ mức 0 đến mức 2 (sử dụng ký pháp Gane-Sarson).

- Thiết kế cơ sở dữ liệu: Sơ đồ thực thể mối quan hệ (ERD) và các lược đồ quan hệ.

- Tài liệu: Báo cáo kỹ thuật chuẩn hóa theo quy định của nhà trường.
